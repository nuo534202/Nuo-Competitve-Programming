# Codeforces Round 1084 (Div. 3)

Contest Link: [Codeforces Round 1084 (Div. 3)](https://codeforces.com/contest/2200)

## C. Specialty String

可以发现将 $s_i$ 和 $s_j$ 替换成 `*` 的操作等价于直接把这两个字符删掉，所以问题就变成了将所有相邻的相同字符删去，如果最后剩下的字符串为空就输出 `YES`，否则输出 `NO`。

时间复杂度： $O(n)$。

## D. Portal

可以将排列按照 Portal 的位置分割成左、中、右三个部分。

- 中间部分可以进行的操作是，将最左边的数字放到最右边，或者将最右边的数字放到最左边，但这两个操作是等价的的，所以我们可以将中间部分看成一个循环队列，因此中间部分的最优排列方式就是将循环队列调整到以最小的数字开头。
- 左右两部分可以进行的操作是，将左边部分的最右边数字放到右边部分的最左侧，或者将右边部分的最左边数字放到左边部分的最右侧。我们目的是让字典序更小，所以我们要让左边部分的所有数字都小于中间部分的最左边的数字（也是最小数字）。因此只要把左边部分所有大于中间部分的最小数字放到右边，或者左边部分原本就是全部小于中间部分的最小数字时，那么就将右边部分左边所有小于中间部分的最小数字放到左边。

时间复杂度： $O(n)$。

## E. Divisible Battle

- 如果序列原本就是非递增的，输出 `Bob`。
- 如果序列其中一个合数有至少两个不同的质因数，那么只需要将较小的质因数 $p$ 放在后面，然后将 $\frac{a_i}{p}$ 放在前面，这样 $\frac{a_i}{p}$ 里面一定会包含另一个更大的质因数，所以无法构成非递增序列，输出 `Alice`。
- 如果所有合数只有的全部质因数都相同，那就看这些质因数本身是不是非递增的，如果是非递增的输出 `Bob`，否则输出 `Alice`。

时间复杂度： $O(n \sqrt{a_i})$。
